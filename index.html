¿Qué es Java?

Java es un lenguaje de programación y una plataforma informática versátil, concurrente, orientada a objetos y basada en clases. Es conocido por su principio **"Escribe una vez, ejecuta en cualquier lugar"** (WORA, por sus siglas en inglés), lo que significa que el código compilado puede ejecutarse en todas las plataformas que soporten Java sin necesidad de recompilación. Esto lo convierte en una opción ideal para el desarrollo de aplicaciones empresariales, móviles (Android), web y de escritorio.

La **Máquina Virtual de Java** (JVM) es el corazón de la plataforma. La JVM interpreta el código de bytes de Java y lo traduce a código nativo de la máquina, lo que permite su portabilidad. Además, Java cuenta con un robusto sistema de gestión de memoria y una amplia biblioteca de APIs (Interfaces de Programación de Aplicaciones).

Conceptos Clave de Java

  • Clases y Objetos

    La base de la programación orientada a objetos (POO). Una clase es un plano para crear objetos, y un objeto es una instancia de una clase.

  • Herencia

    Permite que una clase (subclase) herede propiedades y comportamientos de otra clase (superclase), promoviendo la reutilización de código.

  • Polimorfismo

    La capacidad de un objeto de tomar muchas formas. En Java, esto se logra a través de la sobrecarga y la anulación de métodos.

  • Encapsulamiento

    Oculta los datos y los métodos dentro de un objeto, exponiendo solo una interfaz controlada. Se logra con modificadores de acceso como `public`, `private` y `protected`.
